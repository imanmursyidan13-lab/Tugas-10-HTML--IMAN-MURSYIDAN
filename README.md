## 📌 Deskripsi
Project ini dibuat untuk memenuhi tugas pembelajaran HTML dasar.  
Berisi 10 tag HTML lengkap dengan fungsi dan contoh penggunaannya.

---

## 🧩 Penjelasan Tag HTML

1. `<p>` → Paragraf  
2. `<a>` → Link  
3. `<img>` → Gambar  
4. `<ul>` & `<li>` → List  
5. `<form>` → Form input  
6. `<div>` → Container  
7. `<span>` → Teks inline  
8. `<h1>-<h6>` → Heading  
9. `<button>` → Tombol  
10. `<table>` → Tabel  

---
