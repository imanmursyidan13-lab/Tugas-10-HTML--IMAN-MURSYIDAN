# 📘 Tugas HTML 10 Tag 
🔹 1. <p> (Paragraf)

Fungsi:
Digunakan untuk membuat teks paragraf.

Contoh kode:

<p>Ini adalah contoh paragraf.</p>

Penjelasan:
Tag <p> akan menampilkan teks dalam bentuk paragraf dengan jarak otomatis antar baris.

🔹 2. <a> (Link / Hyperlink)

Fungsi:
Digunakan untuk membuat link ke halaman lain atau website.

Contoh kode:

<a href="https://google.com">Klik ke Google</a>

Penjelasan:

href = tujuan link
Teks di antara <a> adalah yang bisa diklik
🔹 3. <img> (Gambar)

Fungsi:
Menampilkan gambar di halaman web.

Contoh kode:

<img src="https://via.placeholder.com/150" alt="Contoh Gambar">

Penjelasan:

src = sumber gambar
alt = teks jika gambar tidak muncul
🔹 4. <ul> dan <li> (List)

Fungsi:
Membuat daftar (list).

Contoh kode:

<ul>
    <li>HTML</li>
    <li>CSS</li>
</ul>

Penjelasan:

<ul> = daftar tidak berurutan
<li> = isi daftar
🔹 5. <form> (Form Input)

Fungsi:
Membuat form untuk input data pengguna.

Contoh kode:

<form>
    Nama: <input type="text">
    <button>Kirim</button>
</form>

Penjelasan:
Digunakan untuk mengisi data seperti nama, email, dll.

🔹 6. <div> (Container)

Fungsi:
Membungkus atau mengelompokkan elemen HTML.

Contoh kode:

<div>
    Ini adalah div
</div>

Penjelasan:
Biasanya dipakai untuk layout atau pengaturan desain.

🔹 7. <span> (Inline Container)

Fungsi:
Membungkus teks kecil untuk styling.

Contoh kode:

<span style="color:red;">Teks merah</span>

Penjelasan:
Digunakan untuk memberi warna atau style pada bagian teks tertentu.

🔹 8. <h1> - <h6> (Heading)

Fungsi:
Membuat judul atau heading.

Contoh kode:

<h1>Judul Utama</h1>
<h2>Sub Judul</h2>

Penjelasan:

<h1> = terbesar
<h6> = terkecil
🔹 9. <button> (Tombol)

Fungsi:
Membuat tombol yang bisa diklik.

Contoh kode:

<button>Klik Saya</button>

Penjelasan:
Biasanya digunakan untuk submit form atau aksi tertentu.

🔹 10. <table> (Tabel)

Fungsi:
Membuat tabel data.

Contoh kode:

<table border="1">
    <tr>
        <th>Nama</th>
        <th>Kelas</th>
    </tr>
    <tr>
        <td>Iman</td>
        <td>XI RPL II</td>
    </tr>
</table>

Penjelasan:

<table> = tabel
<tr> = baris
<th> = header
<td> = isi data
📌 KESIMPULAN

HTML digunakan untuk membuat struktur website.
10 tag ini adalah dasar penting untuk membuat halaman web sederhana seperti:

teks
gambar
link
form
tabel
